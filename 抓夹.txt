这就是你展示给老师看的“终极流水线”操作：

### 🛠️ 第 0 步：终极编译（只需执行一次）
打开**终端 0**，对整个工作空间进行最后一次全面编译，确保所有的 C++ 和 Python 代码都生效：
```bash
cd ~/dummy_ws
colcon build
```
*(只要看到绿色的 `Summary: X packages finished` 就可以关闭这个终端了)*

---

### 🌍 第 1 步：启动物理世界与数字孪生基座
打开**终端 1**，启动 Gazebo 仿真。机械臂会先出现，5 秒后蓝色小方块会自动降落在它前方：
```bash
cd ~/dummy_ws
source install/setup.bash
ros2 launch dummy-ros2_description gazebo.launch.py
```
*(保持这个终端运行，等待画面出现)*

---

### 👁️ 第 2 步：启动 AI 视觉感知大脑
打开**终端 2**，启动我们写的 Python 视觉节点。它会弹出画面，用黄框锁定蓝方块，红准星对准中心，并向全系统广播 3D 坐标：
```bash
cd ~/dummy_ws
source install/setup.bash
cd src
python3 vision_grasp.py
```
*(保持运行，确保你看到了带有 “Target Locked” 绿色坐标跳动的 OpenCV 窗口)*

---

### 🧠 第 3 步：启动 MoveIt 运动规划大脑
打开**终端 3**，启动 MoveIt 后台服务和 RViz。这是机器人的小脑，负责计算如何避障和规划关节角度：
```bash
cd ~/dummy_ws
source install/setup.bash
ros2 launch dummy_moveit_config demo.launch.py
```
*(保持运行，此时在 RViz 里你应该能看到机械臂，且如果在左下角 Add -> TF，能看到锁定在方块上的 `target_object` 坐标系)*

---

### 🦾 第 4 步：发射！执行自动抓取指令
**激动人心的时刻到了！** 打开**终端 4**，运行我们刚刚编译成功的 C++ 执行器程序：
```bash
cd ~/dummy_ws
source install/setup.bash
ros2 run dummy_grasping grasp_executor
```

敲下最后一次回车后，把目光转移到 Gazebo 和 RViz 的画面上！你会看到终端飞速打印出动作日志，机械臂会自动张开抓夹、移动到蓝色方块上方、精准下降并夹住它，最后抬起回到原位。
